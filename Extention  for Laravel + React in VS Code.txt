PHP & Laravel (Backend)

PHP Intelephense → Autocomplete, go to definition, etc.

Laravel Extra Intellisense → Specific for Laravel facades, routes, config.

Laravel Blade Snippets → Blade files (if any).

Laravel Artisan (optional) → Run Artisan commands from VS Code.

ES7+ React/Redux/React-Native Snippets → Fast component snippets + autocomplete.

Prettier → Code formatting.

JavaScript and TypeScript Nightly → Helps with modern syntax.

Path Intellisense → Helps with auto-imports.
